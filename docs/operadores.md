Operadores

Aritméticos
 
+ 	adição
- 	subtração
* 	multiplicação
/ 	divisão
% 	resto da divisão 

Relacionais
 
> 	maior do que 
>= 	maior ou igual a 
< 	menor do que 
<= 	menor ou igual a 
== 	igual a 
!= 	diferente de 

Lógicos
 
 
&&  	and (ou & se operamos com valores boolean) 
|| 	or (ou | se operamos com valores boolean) 
! 	not 

Bit a bit
 
 
>> op1 >> op2  	move os bits em op1 para a direita pela distância op2 
<< op1 << op2  	move os bits em op1 para a esquerda pela distância op2 
>>> op1 >>> op2  	move os bits em op1 para a direita pela distancia op2 (sem sinal) 
& 	and binário 
| 	or binário 
^ 	xor binário 
~  	complemento binário 

 
 

Atribuição
 
 
Operador 	Uso 	Equivalente
= 	op1 = valor 	
		
+= 	op1 += op2 	op1 = op1 + op2 
-= 	op1 -= op2 	op1 = op1 - op2
*= 	op1 *= op2 	op1 = op1 * op2 
/= 	op1 /= op2 	op1 = op1 / op2 
%= 	op1 %= op2 	op1 = op1 % op2
		
&= 	op1 &= op2 	op1 = op1 & op2
|= 	op1 |= op2 	op1 = op1 | op2
^= 	op1 ^= op2 	op1 = op1 ^ op2
<<= 	op1 <<= op2 	op1 = op1 << op2
>>= 	op1 >>= op2 	op1 = op1 >> op2
>>>= 	op1 >>>= op2 	op1 = op1 >>> op2
